# Abstract

This module establishes the application boundary for predictive modeling, discarding deep learning research in favor of scalable, classic machine learning algorithms. The core objective is selecting, tuning, and evaluating tabular frameworks—specifically **Random Forests, Gradient Boosted Trees (XGBoost/LightGBM)**, and **Logistic Regression**—using standard libraries like `scikit-learn`. Essential recall includes matching business scenarios to proper statistical metrics: prioritizing **Precision** to minimize costly false positives, **Recall** to eliminate catastrophic missed detections (false negatives), and **F1-Score** or **PR-AUC** when dealing with severe class imbalances like financial fraud. MLOps concepts are isolated to deployment reliability, validating split strategies (cross-validation) to combat data leakage, and monitoring for feature drift over time.
